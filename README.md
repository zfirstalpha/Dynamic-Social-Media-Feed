##  Project Overview

This application replicates the core features of a modern social media feed, allowing users to:

- View dynamically loaded posts  
- Like posts  
- Comment on posts  
- Share posts  
- Scroll infinitely through the feed  
- Experience a smooth, responsive interface  

Data is fetched efficiently using GraphQL, enabling flexible queries and optimized performance.

---

##  Project Goals

### 1. Dynamic Data Loading
- Fetch posts using GraphQL queries  
- Display loading states and fallback UI  
- Support real-time updates  

### 2. User Engagement
- Like a post with instant UI updates  
- Add and view comment threads  
- Share posts with custom messages  

### 3. Enhanced User Experience
- Smooth transitions and animations  
- Fully responsive design  
- Infinite scrolling or pagination
